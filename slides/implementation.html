The Implementation

The Web-Stack
  • The language used for implementing the framework is Typescript.
  • Add static strong typing to Javascript.
  • Isomorphic language (the same code runs in the server and in the client).
The Web-Stack
  • The front-end UI is made with
  • Facebook's Reactjs library
  • and Material-UI Components that follows Google's Material Design.
The Web-Stack
  • The communication is done in two times.
  • The first one is in HTTP when the client (browser) sends the first request to get page, styles and scripts as every web page does.
  • The second time is in websocket for real-time communication.
The Web-Stack
  • Nodejs as back-end.
  • The storage is a prevalent system, a database in two parts :
    • The event store for storing the system as a sequence of events.
    • The snapshot for an effective fast reload only.
